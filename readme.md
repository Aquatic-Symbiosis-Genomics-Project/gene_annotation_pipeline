# Overview
This is the repository for the ASG Gene Annotation Pipeline as run at the Welcome Trust Sanger Institute.

# Dependencies
- [Braker](https://github.com/Gaius-Augustus/BRAKER)
- [AUGUSTUS](https://github.com/Gaius-Augustus/Augustus)
- [BAMTOOLS](https://github.com/pezmaster31/bamtools)
- [SAMTOOLS](http://www.htslib.org/)
- [GeneMark](http://opal.biology.gatech.edu/GeneMark/)
- [Diamond](https://github.com/bbuchfink/diamond)
- [ProtHint](https://github.com/gatech-genemark/ProtHint)
- [GenomeThreader](https://genomethreader.org/)
- [Spaln](https://github.com/ogotoh/spaln)
- [NCBI BLAST+](https://blast.ncbi.nlm.nih.gov/Blast.cgi)
- [cdbfasta](https://github.com/gpertea/cdbfasta)
- [GUSHR](https://github.com/Gaius-Augustus/GUSHR)
- [STAR](https://github.com/alexdobin/STAR)
